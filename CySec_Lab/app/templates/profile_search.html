FROM users 
WHERE username = '{{ username if username else '[input]' }}'
  • If you registered a user with a crafted username (e.g., myuser'--), the stored value becomes part of this query.
  • The single-quote closure in your stored data can break the query string and inject additional SQL logic.
  • Example attack flow:
    1. Register with username: ' UNION SELECT username, password, 'leaked', salary, role FROM users--
    2. Search for that user on this page.
    3. The stored username closes the query string and appends a UNION to leak credentials.
  • Even if the registration form "escapes" or "sanitizes" input, the stored data is still used raw here.
